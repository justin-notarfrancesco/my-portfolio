## Overview

A production design system built around the discipline of transit wayfinding: a strict
grid, a small set of type sizes, and semantic color roles rather than one-off values.

## What I built

- A token pipeline (Style Dictionary → CSS variables + TS types) so design and code never drift.
- 40+ accessible React components documented in Storybook with interaction tests.
- A theming layer that retints the entire system from nine base "line" colors.

## Outcome

Adopted across three product teams; cut new-feature UI build time by roughly 40%.
